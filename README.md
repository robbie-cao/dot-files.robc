## Installation

1. Clone a copy to your local

    ```
    $ git clone git@github.com:robbie-cao/config-vim.git
    ```

2. Install dependence packages

    - macos
    
    - ubuntu
    
    - cygwin

3. Run install.sh


Enjoy!
